# Chinese Immigrants in California: Environmental and Social Histories

A website project exploring the environmental and social histories of Chinese immigrants in California from the mid-19th century to the present.

## Project Overview

This website examines how Chinese communities shaped and were shaped by land, water, labor, and environmental regulation, focusing on key landscapes including:

- Angel Island (carceral landscape of detention)
- San Francisco and Oakland Chinatowns (urban ecologies)
- Sacramento-San Joaquin Delta (levee building and agriculture)
- Sierra Nevada railroad corridors (mountain ecology transformation)

## Website Structure

The website consists of the following pages:

1. **Home (index.html)** - Project overview, interactive map, timeline, and thematic sections
2. **Immigration, Law, and Exclusion** - Federal and state laws shaping access to land and resources
3. **Angel Island** - Carceral landscape of detention and exclusion enforcement
4. **Labor & Environmental Transformation** - Mining, railroads, fisheries, and ecological impacts
5. **Delta Agriculture** - Levee building, wetland conversion, and farming communities
6. **Railroad Corridors** - Sierra Nevada construction and mountain ecology transformation
7. **Chinatowns** - Urban renewal, environmental health, and community activism
8. **Post-1965 Migration** - New immigrant waves and environmental justice engagement
9. **Conclusion** - Continuities, identity, and environmental stewardship
10. **Resources** - Bibliography, archives, and research resources

## Usage

### Local Development

1. Open `index.html` in a web browser to view the website locally
2. All pages are linked through the navigation menu
3. CSS styling is in `styles.css`

### Customization

- **Colors**: Modify CSS variables in `styles.css` to change the color scheme
- **Content**: Edit HTML files to update text, add images, or modify structure
- **Styling**: Adjust CSS to match your preferred design aesthetic
